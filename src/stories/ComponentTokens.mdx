<h2 class="docs-tokens-h2">
  EtuLabelChip <code class="docs-tokens-code-inline">.etu-label-chip</code>
</h2>

<div class="docs-tokens-callout">
  Размеры и отступы inline-варианта задаются через внутренние переменные <code class="docs-tokens-code-inline">--_etu-chip-*</code> (см. <code class="docs-tokens-code-inline">EtuLabelChip.vue</code>). Для варианта с точкой (<code class="docs-tokens-code-inline">dot</code>) цвет маркера задаётся пропом <code class="docs-tokens-code-inline">dotColor</code> → <code class="docs-tokens-code-inline">--_etu-chip-dot</code>.
</div>
